# 清华大学编译原理课程（2017秋季）

计算机专业主干课， 编译程序（系统）是计算机系统的核心支撑软件， 贯穿程序语言、运行时系统、体系结构， 联系计算机科学和计算机系统的典范。

## 教学目的
掌握编译程序/系统设计的基本原理；掌握“常见”语言机制的实现技术；经历开发一个小型编译程序的主要阶段；自学并使用自动构造工具；加深对计算机系统的理解；会将所学知识灵活应用。

## 先修课程

- 《数据结构》
- 《形式语言与自动机》（optional）
